Summary & Actions

Combined TTCM exception summary and team notification.

📊 Exception Summary
📊
Run at least one report first.
🎫 Raise Jira Ticket — notify the source team of TTCM changes

Create a Jira ticket with the exception summary. The source team can reply confirming which changes are expected or flagging items for investigation. The AI agent reads their replies and provides a recommendation.

🤖
AI Agent

Once the team replies on Jira, click Ask Agent on the ticket card below. Claude will read the comments and provide an APPROVE / INVESTIGATE / ESCALATE recommendation based on the nature of the TTCM changes discussed.
